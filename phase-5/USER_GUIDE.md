## Creating Tasks

### Basic Task Creation

1. Click the **"New Task"** button (+ icon)
2. Enter a **title** (required)
3. Add a **description** (optional)
4. Click **"Create"**

Your task is now created and will appear in your task list!

### Adding Details

When creating a task, you can add:

**Priority Level**:
- Low (default)
- Medium
- High
- Urgent

**Tags**:
- Type a tag name and press Enter
- Add multiple tags for better organization
- Examples: "work", "personal", "urgent", "meeting"

**Due Date**:
- Click the calendar icon
- Select a date and time
- Tasks with due dates will show a countdown

**Description**:
- Add detailed notes about the task
- Supports plain text
- Great for adding context or instructions

### Example: Creating a Work Task

```
Title: Prepare quarterly report
Description: Compile sales data and create presentation for Q4 review meeting
Priority: High
Tags: work, report, quarterly
Due Date: December 31, 2024, 5:00 PM
```

## Recurring Tasks

### What Are Recurring Tasks?

Recurring tasks automatically create a new instance when you complete them. Perfect for:
- Daily standup meetings
- Weekly reports
- Monthly reviews
- Annual evaluations

### Creating a Recurring Task

1. Click **"New Task"**
2. Enter task details
3. Check **"Make this recurring"**
4. Select recurrence pattern:
   - **Daily**: Repeats every day
   - **Weekly**: Repeats every 7 days
   - **Monthly**: Repeats on the same day each month
   - **Yearly**: Repeats annually

5. Click **"Create"**

### How It Works

When you complete a recurring task:
1. The current task is marked as complete
2. A new task is automatically created
3. The new task has the same title, description, priority, and tags
4. The due date is calculated based on the recurrence pattern

### Example: Daily Standup

```
Title: Daily Standup Meeting
Recurrence: Daily
Priority: Medium
Tags: meeting, daily
```

When you complete today's standup, tomorrow's standup is automatically created!

## Search and Filters

### Quick Search

1. Click the search bar at the top
2. Type your search query
3. Results appear instantly

**Search Tips**:
- Searches in task titles and descriptions
- Case-insensitive
- Partial matches work
- Example: "report" finds "Quarterly Report", "Report Bug", etc.

### Filtering Tasks

**By Priority**:
- Click the priority filter dropdown
- Select: Low, Medium, High, or Urgent
- Only tasks with that priority are shown

**By Tags**:
- Click **"Filter by Tags"**
- Select one or more tags
- Tasks with any selected tag are shown

**By Status**:
- **All**: Show all tasks
- **Active**: Show incomplete tasks only
- **Completed**: Show completed tasks only

**By Due Date**:
- **Overdue**: Tasks past their due date
- **Today**: Tasks due today
- **This Week**: Tasks due within 7 days
- **This Month**: Tasks due within 30 days

### Sorting Tasks

Click the sort dropdown to order by:
- **Created Date** (newest first)
- **Due Date** (soonest first)
- **Priority** (highest first)
- **Title** (alphabetical)

### Advanced Filtering

Combine multiple filters:
```
Priority: High
Tags: work, urgent
Status: Active
Sort by: Due Date
```

This shows all high-priority, urgent work tasks that are incomplete, sorted by due date.

## Audit Trail

### What Is the Audit Trail?

The audit trail is a complete history of all changes to your tasks:
- When tasks were created
- What was modified
- When tasks were completed
- Who made the changes

### Viewing Audit Logs

1. Click **"Audit"** in the navigation menu
2. See a chronological list of all operations
3. Filter by:
   - Task
   - Operation type
   - Date range

### Understanding Audit Entries

Each entry shows:
- **Operation**: What happened (created, updated, completed, deleted)
- **Task**: Which task was affected
- **Changes**: What was modified
- **Timestamp**: When it happened
- **User**: Who made the change

### Example Audit Entry

```
Operation: task.updated
Task: Quarterly Report
Changes:
  - Priority: Medium ‚Üí High
  - Tags: Added "urgent"
Timestamp: 2024-12-15 14:30:00
User: john@example.com
```

### Use Cases

**Track Progress**:
- See how long tasks take
- Identify bottlenecks
- Measure productivity

**Accountability**:
- Know who changed what
- Audit for compliance
- Review team activity

**Recovery**:
- See what was changed
- Understand task history
- Restore information if needed
